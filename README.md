# apex

HCMIU Web Application Development Project

## Project Overview
The `apex` project is a comprehensive web application developed as part of the Web Application Development course. It consists of two main components:

1. **Backend**: The backend is built using Python and leverages modern tools and frameworks to provide robust APIs and manage data. For detailed documentation, refer to the [Backend README](./backend/README.md).

2. **Frontend**: The frontend is developed using modern JavaScript frameworks and tools to deliver an interactive and user-friendly interface. For detailed documentation, refer to the [Frontend README](./frontend/README.md).

---

This project demonstrates the integration of a scalable backend with a dynamic frontend to create a seamless user experience.
